# Project Showcase tiles

The 5 tiles in the "Project Showcase" strip on the home page read images from this folder.

Save the photos here with **these exact filenames** (lowercase, hyphenated, `.jpg`):

| Tile | Filename |
| ---- | -------- |
| Architectural Model | `architectural-model.jpg` |
| Structural Model | `structural-model.jpg` |
| MEP Model | `mep-model.jpg` |
| MEP Coordination (Clash Check) | `mep-coordination.jpg` |
| Shop Drawing / Detail | `shop-drawing.jpg` |

Recommended size: 800 × 600 px JPG (4:3), 100–250 KB each.
The tiles use `aspect-[4/3] object-cover` so anything wider than 4:3 will be
center-cropped — frame the subject toward the middle.

If a file is missing, the tile falls back to a procedural SVG illustration
automatically — no broken images.
