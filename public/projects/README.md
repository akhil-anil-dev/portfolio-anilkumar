Drop project screenshots / renders here and reference them by filename in `lib/data.ts` via the `image` field.

- `noru-hotel.jpg`
- `hyatt-centric.jpg`
- `gasan-mamo.jpg`
- `sarah-gasan-villa.jpg`
- `park-hyatt-riyadh.jpg`
- `address-the-bay.jpg`

Until real images are added, the project cards show a blueprint-style placeholder.
To wire images up, open `components/Projects.tsx` and uncomment / swap the placeholder block
for an `<img>` tag — there's a TODO note inline.

Recommended size: 1600 × 1000 px, JPG or WebP, ~250 KB each.
